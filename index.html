CSS Power-ups

An eight-week course on everything CSS.

Week 1

Topic

Introduction and selectors

Main Points

  • Rules, selectors, and properties.
  • Pseudo-selectors.
  • How selectors relate to HTML structure.

Challenge

CSS selector game — see who can finish first!

Week 2

Topic

The cascade and specificity

Main Points

  • Specificity problems and how to avoid them.
  • How to debug specificity wars.

Challenge

Specificity quiz — see who can finish first!

Week 3

Topic

The box model and positioning

Main Points

  • Block elements vs. inline elements.
  • How to center things vertically.
  • Margin, padding, and other layout properties.

Challenge

Implement this Lucidchart wireframe in CSS (ignoring text).

Week 4

Topic

Property categories and CSS3 modules

Main Points

  • Pseudo-elements.
  • All the cool things that can be done with CSS these days.
  • How to check for browser compatibility.

Challenge

Implement the coding challenge!

Week 5

Topic

Responsive CSS

Main Points

  • Media queries.
  • Non media-query solutions.

Challenge

Make your project from last week scale down to mobile nicely.

Week 6

Topic

CSS preprocessors

Main Points

  • Advantages and disadvantages of preprocessors generally.
  • Strengths and weaknesses of the most popular preprocessors.

Challenge

Refactor your solution from last week using a preprocessor.

Week 7

Topic

Performance and Chrome's rendering cycle

Main Points

  • How to make sure your CSS is performant.
  • How to navigate Chrome's DevTools to this regard.

Challenge

Implement a parallax scroll effect, and make it as performant as you can.

Week 8

Topic

Frameworks and architecture paradigms

Main Points

  • Atomic, OOCSS, BEM, and other paradigms.
  • Overviews of Compass, Bootstrap, and other frameworks.

Challenge

Implement a small CSS framework that allows end users to style their typography. Use any approach you like.
